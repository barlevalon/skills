# release-prep

Evidence-backed release preparation for software repos.

Use this skill when preparing releases, choosing SemVer bumps, drafting changelogs or release notes, validating release readiness, replacing commit-driven release automation, or creating project-local release policy.

## Install

```bash
pi install npm:@barlevalon/release-prep-skill
```

Local checkout:

```bash
pi install ./skills/release-prep
```
